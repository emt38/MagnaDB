About

MagnaDB is intended to provide high-end ORM functionalities while providing a certain degree of freedom to the developer. It's intended to be malleable, while remaining safe to use.

MagnaDB is also designed to use as fewer resources as possible, while also remaining fast and reliable at any given time.

This project started years ago, it was originally a smaller and private library, and it didn't have many of the functionalities that it now has, but then set out to be a public framework since it proved to be quite useful.
